# Example

## Env

- Node 16

## Setup
```sh
$ npm install
```
## Start
Open two terminals.
Go to the example directory.
Start Client :
```sh
$ npm run dev
```
Start Server :
```sh
$ npm start
```

## Use case

Open a window 1 on http: // localhost: 3000 /
Create a room 1.
Open another window 2 on http: // localhost: 3000 /
Join room 1.
You can chat.
Open another window 3 on http: // localhost: 3000 /
Create a room 2.
Leave the room of window 2 and go to room 2
You can chat.
